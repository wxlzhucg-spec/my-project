# 🎮 Pandas数据分析闯关模式

一个交互式的Pandas学习系统，通过8个关卡全面掌握数据分析核心技能。

## 🚀 快速开始

### 1. 安装依赖
```bash
cd /Users/xl/Desktop/xinyu_v1/pandas_training
pip install -r requirements.txt
```

### 2. 生成模拟数据
```bash
python data_generator.py
```
这将生成三个CSV文件：
- `orders.csv` - 订单数据（1000条记录，包含缺失值和异常值）
- `users.csv` - 用户数据（200条记录）
- `products.csv` - 产品数据（50条记录）

### 3. 开始闯关
```bash
python pandas_challenge.py
```

## 🎯 关卡介绍

### 第1关：数据加载与探索
- 读取CSV文件
- 查看数据基本信息
- 检查缺失值和重复值
- **积分：10分**

### 第2关：数据清洗
- 处理缺失值（用类别平均价格填充）
- 删除无效日期记录
- 修正异常值（负数量）
- 优化数据类型
- **积分：15分**

### 第3关：数据合并与关联
- 合并订单、用户、产品数据
- 计算总金额（price × quantity）
- 提取日期信息（年、月、日、星期几）
- **积分：15分**

### 第4关：数据分组与聚合
- 按城市统计订单和销售额
- 按产品类别分析价格和评分
- 按性别分析消费行为
- 使用agg计算多维度统计
- **积分：20分**

### 第5关：时间序列分析
- 设置时间索引
- 按周/月重采样统计
- 计算滚动平均销售额
- **积分：20分**

### 第6关：用户行为分析
- 找出TOP 10消费用户
- 计算用户平均订单金额
- RFM模型分析（最近购买时间、频率、金额）
- 用户分层（8个层级）
- **积分：20分**

### 第7关：数据可视化
- 绘制每日销售额折线图
- 绘制城市销售额柱状图
- 绘制产品类别占比饼图
- 绘制用户年龄分布直方图
- **积分：15分**

### 第8关：性能优化与高级技巧
- 优化内存使用（category类型）
- 使用query优化查询
- 使用eval表达式求值
- 向量化操作替代循环
- **积分：15分**

## 🎮 游戏玩法

### 基本命令
- `check` - 检查当前关卡任务完成情况
- `hint` - 查看详细提示和示例代码
- `skip` - 跳过当前关卡（不获得积分）
- `quit` - 退出游戏（进度自动保存）

### 进度系统
- 进度自动保存在 `challenge_progress.json`
- 每关通过获得相应积分
- 可以随时退出，下次继续

### 代码编写提示
1. 每个关卡需要编写Python代码完成任务
2. 可以使用示例代码作为参考
3. 重要：第3关完成后，需要将合并结果赋值给 `self.combined_df` 供后续关卡使用
4. 可视化关卡需要安装matplotlib和seaborn

## 📊 数据集说明

### 订单数据 (orders.csv)
- 1000条订单记录
- 字段：order_id, user_id, product_id, order_date, price, quantity, status
- 包含缺失值和异常值（用于数据清洗练习）

### 用户数据 (users.csv)
- 200条用户记录
- 字段：user_id, username, city, age, gender, register_date
- 包含分类变量和数值变量

### 产品数据 (products.csv)
- 50条产品记录
- 字段：product_id, product_name, category, cost, rating
- 包含产品类别、成本、评分等信息

## 🔧 故障排除

### 常见问题
1. **数据文件不存在**
   - 运行 `python data_generator.py` 生成数据

2. **导入pandas失败**
   - 确保已安装：`pip install pandas numpy`

3. **中文字符显示问题**
   - 系统自动尝试配置中文字体
   - 如仍有问题，可修改代码中的字体设置

4. **可视化无法显示**
   - 确保安装了matplotlib：`pip install matplotlib seaborn`
   - 图表会自动保存为图片文件

5. **进度丢失**
   - 进度自动保存在 `challenge_progress.json`
   - 不要手动删除此文件

### 环境要求
- Python 3.7+
- pandas >= 2.0.0
- numpy >= 1.24.0
- matplotlib >= 3.7.0（可选，用于可视化）
- seaborn >= 0.12.0（可选，用于高级可视化）
